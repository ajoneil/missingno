# Test ROM Attribution

These test ROMs are used to verify Game Boy emulation correctness.
They were obtained from the [c-sp/game-boy-test-roms](https://github.com/c-sp/game-boy-test-roms) collection (v7.0).

- **Author:** Shay Green (blargg)
- **Source:** https://github.com/retrio/gb-test-roms
- **License:** No explicit license. Widely redistributed across emulator projects.
